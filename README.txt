This application was developed using Unity 2019.1.4f1. 

The Executable does not require Unity to be installed, and it can be run 
by opening "root\Executable\Unity Planet Project.exe".

Application Features:
	- Ability to smoothly travel from planet to planet bhy pressing each planet button
	- When focused on a planet, user is able to zoom in/out using the mouse scroll wheel
	- Ability to view solar system from a distance
	- Ability to travel to the next procedurally generated solar system by pressing the "Hyperdrive" button. After the 
	  button is pressed, user will need to wait 3-4 seconds for the solar system to load, and can go to the newly 
	  created solar system by pressing the button once more.

Unity Inspector Options can be viewed by opening the project and selectingt a Planet object within a planet container.
